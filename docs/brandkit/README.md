## At a glance

- **Primary colour:** Sky Blue `#0E72B4` (supports white text — 5.14:1 AA)
- **Secondary colour:** Control Navy `#15324F` (nav, headers, wordmark)
- **Icon story:** start → control (ATC tower) → cleared (green check), with a climbing aircraft and the Swiss cross
- **Fluent v9:** `import { atcSimulatorLightTheme } from './color/atcsimulator-theme'`

SVG files are the scalable masters; PNGs are convenience previews.
