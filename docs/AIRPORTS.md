> **How to read this document.** Coordinates are the **airport reference point** in **decimal degrees (WGS84)** — latitude (N+) then longitude (E+). "Flight code" = the **ICAO** location indicator (4 letters, all Swiss codes start with **LS**) and, where the airport has one, the **IATA** code (3 letters). Values were verified from each airport's reference page; for operational/precision use, always confirm against the official **Swiss AIP** (skyguide / BAZL–FOCA).

## Sources & method

- **ICAO / IATA codes, names, cantons, categories:** Wikipedia *List of airports in Switzerland* (EN) and *Liste der Schweizer Flugplätze* (DE), cross-referenced with OurAirports and the Swiss aerodrome register (BAZL/FOCA). Retrieved **17 July 2026**.
- **Coordinates (Sections 1–4):** decimal degrees (WGS84), airport reference point, taken from each airport's Wikipedia reference page (sourced there to the Swiss AIP / EUROCONTROL / DAFIF).
- Coordinates are rounded to ~5 decimal places (≈ 1 m). For flight planning, navigation, or any operational use, **verify against the current official Swiss AIP** (skyguide / BAZL) — this document is a reference compilation, not an aeronautical source.
- Switzerland's ICAO prefix is **LS**. The EuroAirport (LFSB) is the exception — it lies on **French** soil, so it carries a French **LF** code while serving Basel.
